◆ CriterionType

enum GMDH::CriterionType
strong

Enum class for specifying the external criterion \( E \) to select the optimum solution.

Table of symbols:
\( A \) - training data;
\( B \) - testing data;
\( C \) - learning data that is equal to \( A\cup{B} \);
\( X_A \) - input variables matrix of the training data \( A \);
\( X_B \) - input variables matrix of the testing data \( B \);
\( X_C \) - input variables matrix of the learning data \( C \) that is equal to \( X_A\cup{X_B} \) ;
\( y_A \) - target values vector of the training data \( A \);
\( y_B \) - target values vector of the testing data \( B \);
\( y_C \) - target values vector of the learning data \( C \) that is equal to \( y_A\cup{y_B} \);
\( \hat{w}_A \) - coefficients vector of the model trained on a training data \( A \);
\( \hat{w}_B \) - coefficients vector of the model trained on a testing data \( B \);
\( \hat{w}_C \) - coefficients vector of the model trained on a learning data \( C \);

Enumerator
regularity 

\( E=||y_{B}-X_{B}\hat{w}_A||^2 \)

symRegularity 

\( E=||y_{B}-X_{B}\hat{w}_A||^2+||y_{A}-X_{A}\hat{w}_B||^2 \)

stability 

\( E=||y_{C}-X_{C}\hat{w}_A||^2 \)

symStability 

\( E=||y_{C}-X_{C}\hat{w}_A||^2 + ||y_{C}-X_{C}\hat{w}_B||^2 \)

unbiasedOutputs 

\( E=||X_{B}\hat{w}_A-X_{B}\hat{w}_B||^2 \)

symUnbiasedOutputs 

\( E=||X_{C}\hat{w}_A-X_{C}\hat{w}_B||^2 \)

unbiasedCoeffs 

\( E=||\hat{w}_A-\hat{w}_B||^2 \)

absoluteNoiseImmunity 

\( E=(X_B\hat{w}_C-X_B\hat{w}_A)^T(X_B\hat{w}_B-X_B\hat{w}_C) \)

symAbsoluteNoiseImmunity 

\( E=(X_C\hat{w}_C-X_C\hat{w}_A)^T(X_C\hat{w}_B-X_C\hat{w}_C) \)

◆ Solver

enum GMDH::Solver
strong

Enum class for specifying the QR decomposition method for linear equations solving in models.

Detailed description of methods can be found here

Enumerator
fast 

Fast solution with perhaps not the best accuracy using HouseholderQR decomposition.

accurate 

Slow solution with maximum accuracy using FullPivHouseholderQR decomposition.

balanced 

Balanced solution with average speed and accuracy using ColPivHouseholderQR decomposition.

Validate input parameters values.

Parameters
testSizeFraction of the input data that should be placed into the second part
pAverageThe number of best models based of which the external criterion for each level will be calculated
threadsThe number of threads used for calculations. Set -1 to use max possible threads
verbose1 if the printing detailed infomation about training process is needed, otherwise 0
limitThe minimum value by which the external criterion should be improved in order to continue training
kBestThe number of best models based of which new models of the next level will be constructed
Exceptions
std::invalid_argumentif testSize, pAverage, limit or kBest value is incorrect
Warning
If the threads or verbose value is incorrect an exception won't be thrown. Insted, the incorrect value will be replaced with the default value and a corresponding warning will be displayed
Returns
Method exit status
