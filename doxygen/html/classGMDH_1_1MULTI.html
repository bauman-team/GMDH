Fit the algorithm to find the best solution.

Parameters
xMatrix of input data containing predictive variables
yVector of the taget values for the corresponding x data
criterionSelected external criterion
kBestThe number of best models based of which new models of the next level will be constructed
testSizeFraction of the input data that should be used to evaluate models at each level
pAverageThe number of best models based of which the external criterion for each level will be calculated
threadsThe number of threads used for calculations. Set -1 to use max possible threads
verbose1 if the printing detailed infomation about training process is needed, otherwise 0
limitThe minimum value by which the external criterion should be improved in order to continue training
Exceptions
std::invalid_argument
Warning
If the threads or verbose value is incorrect an exception won't be thrown. Insted, the incorrect value will be replaced with the default value and a corresponding warning will be displayed
Returns
A reference to the algorithm object for which the training was performed
